# 📊 Telecom X - Análisis de Evasión de Clientes

## 📌 Descripción del Proyecto

Este proyecto tiene como objetivo analizar el comportamiento de los clientes de Telecom X para identificar los factores que influyen en la cancelación del servicio (Churn).

La evasión de clientes representa un desafío importante para las empresas de telecomunicaciones, ya que impacta directamente en los ingresos y en el crecimiento del negocio. A través del análisis de datos, se busca identificar patrones que permitan desarrollar estrategias para mejorar la retención de clientes.

---

## 🎯 Objetivo

Analizar los datos de clientes de Telecom X para identificar las variables que influyen en la cancelación del servicio y generar insights que ayuden a reducir la tasa de churn.

---

## 🛠 Tecnologías Utilizadas

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

Estas herramientas fueron utilizadas para la limpieza de datos, análisis exploratorio y visualización de información.

---

## 📊 Proceso de Análisis

El proyecto se desarrolló en las siguientes etapas:

1. Extracción de datos desde una API en formato JSON
2. Conversión de los datos a un DataFrame de Pandas
3. Limpieza y tratamiento de datos
4. Análisis exploratorio de datos (EDA)
5. Visualización de patrones relacionados con la evasión de clientes
6. Generación de conclusiones y recomendaciones estratégicas

---

## 📈 Principales Hallazgos

A partir del análisis realizado se identificaron varios factores relacionados con la evasión de clientes:

- Los clientes con **contratos mensuales presentan mayor probabilidad de cancelar el servicio**.
- Los clientes con **mayores cargos mensuales tienden a abandonar la empresa con mayor frecuencia**.
- Los clientes con **menor tiempo de permanencia en la empresa presentan mayor tasa de churn**.
- Algunos **métodos de pago están asociados a mayor cancelación**.

---

## 💡 Recomendaciones

Basado en los resultados del análisis, se sugieren las siguientes acciones:

- Promover contratos de mayor duración mediante incentivos.
- Implementar programas de fidelización para clientes nuevos.
- Revisar los planes con cargos mensuales más altos.
- Analizar los métodos de pago asociados con mayor churn.
- Desarrollar modelos predictivos para anticipar cancelaciones.
